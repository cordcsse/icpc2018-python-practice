Input files of the form "###-n-m-c.in" (e.g., 005-100-100-50.in) indicate
data set number ###, n (number of relationships), m (number of queries), 
and c (number of distinct names appearing in relationships). 

E.g., data set 009-10000-10000-500.in has 10,000 relationships, 10,000
queries, and 500 distinct names.
